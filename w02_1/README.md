# 실습과제 1
## 전처리 또는 선행처리 단계를 설명하라
- 전처리란 컴파일이 진행되기전 소스코드를 미리 처리하는 과정이다.
## 객체 cin, cout은 어디에 선언되어 있는가? 선언된 파일을 찾아서 의미를 설명하고 캡쳐하여 첨부하라 
- C++ 라이브러리 iostream 안에 선언되어있다. cout 은 표준 출력 스트림 객체이고 cin 은 표준 입력 스트림 객체이다.
## 프로그램을 링크할때 이름 충돌이 발생하는 경우를 설명해보라
- 여러명이 하나의 프로젝트를 작업하는경우 충돌할수 있다.
## 스트림이란 무엇인가?
- 스트림이란 데이터가 이동하는 통로이다. 
## 표준 출력 스트림을 정의할 때 사용하는 C언어 자료형은?
- FILE 자료형이다.
## 표준 출력 스트림을 정의할 때 사용하는 C++언어 자료형은?
- ostream 자료형이다.
